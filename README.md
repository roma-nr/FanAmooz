# فن‌آموز (FanAmooz)

سامانهٔ مدیریت یادگیری (LMS) برای دانشکده‌های ملی مهارت

## نصب سریع
۱. پروژه را در `htdocs` قرار دهید.
۲. دیتابیس `fanamooz` را با اجرای `sql/fanamooz.sql` بسازید.
۳. فایل `config/database.php` را با اطلاعات دیتابیس خود ویرایش کنید.
۴. (اختیاری) `sample_data.sql` را برای داده‌های نمایشی import کنید.
۵. در مرورگر به `http://localhost/FanAmooz` بروید.
